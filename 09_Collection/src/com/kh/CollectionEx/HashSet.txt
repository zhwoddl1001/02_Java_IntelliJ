중복 데이터를 허용하지 않고, 저장된 요소의 순서를 유지하지 않는 자료구조
데이터를 효율적으로 저장하고 검색하기 위해 사용됨

주요 특징
1.동일한 값이 이미 존재하면 추가되지 않음

2.저장된 순서를 유지하지 않고, 내부적으로 Hashing 기법 사용

3.null 값 허용 = 아무것도 없는 값을 저장하는 것을 허용
        아무것도 없는 것도 하나의 값이라고 생각하기 때문
        대신 최초 1회만 저장할 수 있음


ex)바지주머니 물건을 넣는 것(= 순서대로 물건을 넣는다 하더라도 순서가 의미 없음
            손에 집히는대로 필요한대로 바지주머니 안에 있는 물건을 꺼낼 것이기 때문
            대신 중복된 물건을 넣을 수 없는 주머니



주요 메서드(기능)

add(E e)               : 리스트 끝에 요소(=값)를 추가

remove(Object O )      : 지정된 요소를 삭제(첫 번째로 발견된 값만 삭제)
    ┖─────────────────── 만약에 값이 정수형일 때는 remove(Integer.valueOf(정수값));
size()                 : 리스트의 크기를 반환
isEmpty()              : 리스트가 비어있는지 확인
contains(object O)     : 리스트에 특정 요소가 포함되어 있는지 확인
clear()                : 리스트의 모든 요소를 삭제
addAll(Collection c)   : 다른 컬렉션의 모든 요소를 복사해서 추가
iterator()             : Iterator를 반환해 순차적으로 회전 (for문 대신 이용)가능
